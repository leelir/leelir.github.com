1.CocoaPod 的安装使用

由于某些原因，下面这两句就不解释了: 

$ gem sources --remove https://rubygems.org/
//等有反应之后再敲入以下命令
$ gem sources -a http://ruby.taobao.org/

完成以后执行： 

$ sudo gem install cocoapods

好啦安装完成了！
下面再说一下如何使用，首先要在项目目录里面通过 

pod init

创建一个podfile文件。
在podfile文件里面添加代码以Predict项目中引用的Material为例：

target 'Predict' do
source 'https://github.com/CocoaPods/Specs.git'
use_frameworks!
pod 'Material', '~> 1.40.1'
end

最后执行 

pod install

完毕以后将framework添加到项目中去。
Cocoapod尽量安装稳定版本 如果因为更新了新的版本而出现了问题，可以考虑降低版本。
首先移除安装包： 

$which pod
$sudo rm-rf<path>

然后移除 RubyGems 中的 Cocoapods程序包： 

$gem list
$sudo gem uninstall cocoapods-v xxxx.x.x
$sudo gem uninstall cocoapods-core-v xxxx.x.x

最后按照指定版本的cocoapod： 

$ gem
 install cocoapods --user-install -v xxx.xx.x
